Allgemeiner Aufbau, Struktur, grob:

Eine ZENTRALE verwaltet alles:
-   Verbindet sich mit dem IRC -> Instanz der IRCInterface-Klasse mit 2 Threads dafür.
    Dabei leitet sie die empfangen nachrichten bei bedarf an fensterfunktionen weiter.
    Abgehende Nachrichten werden an die entsprechenden benutzer/räume gesendet
-   Sie verwaltet die verbindung zur konfigurationsschnittstelle und wendet 
    dort festgelegte einstellungen an oder übergibt geänderte einstellungen
-   Sie erzeugt bei Bedarf (beim Start) neue Instanzen der Fensterklasse

INSTANZEN/FENSTER:
-   Jeder Raum wird in einem eigenen Fenster (neuer Instanz der Userinterface-Klasse)
    dargestellt
-   Nachrichten an einen selber, werden im aktuellen "obersten" Fenster angezeigt


die fensterklasse enthält funktionen die generell nur aus der zentrale verwaltet werden.

Folgende Funktionen werden gebraucht:
    müssen Ausgabe lokalisiert und angepasst ausgeben, dazu verbinden sie siche mit 
    einer Konfigurationsverarbeitetenfunktion und übergeben bis zu VIER, 4 Parameter

Titel ändern
Thema ändern
Benutzerliste bearbeiten (komplett, hinzufügen, entfernen, status/modus ändern, name ändern) (auto whois?)
Nachricht in bestimmten Fenster anhängen
    
    muss Ausgabe lokalisiert und angepasst ausgeben, dazu mit Konfiguration verbinden
    und Parameter übergeben



Konfigurationsschnittstelle:
    Anforderungen:
        - Jede Option hat einen voreingestellten Wert
        - Lokale oder entfernte (protokoll?) Konfigurationsdatei
        - Format der Konfigurationsdatei? -> wie INF?
            [ZENTRALE] ... [IRC] ... [FENSTER]
        - mit folgenden optionen:
            - Änderungen in Konfigdatei speichern
            - Farben
            - Anzahl der erlaubten Fenster und damit maximale Anzahl an Räumen
            - Ausgabeformat von Text ( <nick> nachricht / [nick] nachricht / zeitstempel / ...)
            - Lokalisierung
        - wxString.Replace...
        - Verarbeitenfunktion gibt bei Bedarf lokalisierten und angepassten Text zurück
